# :book: emojilib in spanish 

Emoji keyword library in spanish

## Install

### :bird: Bower

```
bower install emojilib-spanish
```

### :package: NPM

```
npm install emojilib-spanish --save
```

```javascript
var emojis = require("emojilib-spanish")
```
